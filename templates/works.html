02 : Works

These projects demonstrate my proficiency in Data Science, encompassing Data Analysis, Machine Learning, Deep Learning, and statistical modeling. Each project reflects my ability to leverage data-driven techniques to solve real-world problems across various domains, including Predictive Modeling, Data Visualization, and beyond. I invite you to explore my work to see how I approach complex challenges with innovative solutions and a strong analytical mindset.

001/006

Multi Class Dog Breed Identification

This project demonstrates building an end-to-end multi-class image classifier using TensorFlow 2.0 and TensorFlow Hub. The task is to identify the breed of a dog given an image, which involves deep learning techniques and transfer learning.

Github

002/006

People Personality Prediction

This project aims to predict an individual's MBTI (Myers-Briggs Type Indicator) personality type based on a set of demographic and interest-related features. The data is synthetic and designed for personality research purposes, with over 100,000 samples and various input features, including age, gender, education level, and personality-related scores.

GitHub

003/006

Bulldozer Price Prediction

This contains an end-to-end machine learning project to predict the sale price of bulldozers. The project uses historical auction data and other relevant features to train machine learning models that predict the final auction sale prices of bulldozers.

Github

004/006

Heart Disease Classification

This project leverages machine learning algorithms to predict whether a patient has heart disease based on various medical attributes. The dataset used comes from the Cleveland Heart Disease data

Github

005/006

Image Recognition

This project is all about teaching a computer to see! We'll use a cool library called Image AI to build a program that can figure out what's going on in a picture.

Github

006/006

Password Checker

This project offers users a safe and easy way to check the security of their passwords without having to worry about their passwords being stored in a central database. It gives users peace of mind knowing that their passwords are being checked securely on their own computer.
